# Card-R-W-Tools

Card-R-W-Tools是一款用于处理NFC卡片（特别是Bambu系列耗材卡片）的Android应用工具，支持卡片读取、信息展示、数据存储、导入导出等功能。

## 功能特点

- **NFC卡片读取**：支持读取Mifare Classic类型卡片数据，解析Bambu耗材卡片信息
- **卡片信息展示**：显示耗材类型、颜色、生产日期、尺寸规格、温度参数等详细信息
- **数据管理**：本地存储卡片数据，支持新增、删除、覆盖保存操作
- **数据导出**：将卡片数据导出为.bin、.mct、.keys格式文件至下载目录
- **卡片写入**：支持将存储的卡片数据写入新的FUID卡片（需注意每张卡片仅可写入一次）
- **权限管理**：适配不同Android版本的存储权限请求机制

## 技术栈

- 语言：Kotlin
- 构建工具：Gradle (KTS)
- 架构组件：ViewBinding、Coroutines
- 数据存储：SQLite
- NFC处理：Android NFC API、MifareClassic
- 第三方库：
  - BouncyCastle (加密处理)
  - Gson (数据处理)
  - AndroidX组件

## 环境要求

- Android Studio 2023.1+
- Gradle 8.13
- Android SDK：
  - 最低支持版本：Android 9.0 (API 28)
  - 目标版本：Android 14 (API 36)
- JDK 21

## 安装方法

1. 克隆本项目到本地
   ```bash
   git clone <项目仓库地址>
   ```
2. 使用Android Studio打开项目
3. 等待Gradle同步完成
4. 连接Android设备并运行应用

## 使用说明

1. **首次启动**：应用会展示使用条款，确认后进入主界面
2. **读取卡片**：
   - 确保设备NFC功能已开启
   - 将卡片贴近设备NFC感应区
   - 应用会自动读取并解析卡片信息
3. **保存卡片**：读取完成后可选择保存卡片信息到本地
4. **导出数据**：通过菜单中的"导出"功能将卡片数据保存到下载目录
5. **写入卡片**：在卡片详情页点击"创建新卡片"，然后将新卡片贴近感应区完成写入

## 注意事项

- 本应用需要设备支持NFC功能
- 写入功能仅支持FUID类型卡片，且每张卡片仅可写入一次
- 对于Android 10以下设备，需要手动授予存储权限
- 导出的文件位于`Download/BCardTools/<时间戳>/`目录下
